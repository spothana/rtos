# RTOS Architectural Overviews & Visualizations

Welcome to the **RTOS** repository. This project provides a comprehensive visual guide and architectural overview of popular Real-Time Operating Systems, specifically focusing on **FreeRTOS** and **Zephyr**.

The repository contains high-quality diagrams and visualizations designed to help developers, students, and systems architects understand the internal structures, kernel components, and operational flows of these RTOS platforms.

## Repository Structure

The contents are organized into dedicated directories for each RTOS:

### 1. [FreeRTOS](./freertos)
This directory contains detailed visualizations of the FreeRTOS kernel and its architecture.
- **Contents:** 13 architectural diagrams (`FreeRTOS1.png` through `FreeRTOS13.png`).
- **Focus:** Task management, scheduling mechanisms, and core kernel components.

### 2. [Zephyr](./zephyr)
This directory provides comprehensive visual overviews of the Zephyr Project's modular ecosystem.
- **Contents:** 15 detailed diagrams (`Zephyr1.png` through `Zephyr15.png`).
- **Focus:** Modular architecture, kernel services, and hardware abstraction layers.

## How to View and Use

These resources are designed to be used as a reference for learning, teaching, or technical documentation.

- **Browsing:** Navigate to the `freertos/` or `zephyr/` directories and click on any PNG file to view the visualization directly in GitHub.
- **Usage:** These diagrams are ideal for use in technical presentations, study materials, or as visual aids when performing architectural reviews of embedded systems.
